Вертикализатор для взрослых REAFIT Aidflex X3 BASIC

Вертикализатор для взрослых AIDFLEX X3 BASIC - поднимает человека непосредственно из кресла-каталки, кровати или другой поверхности, переводя его в вертикальное положение. Это позволяет людям, испытывающим затруднения, принимать вертикальное положение как для профилактики и выполнения упражнений, так и для улучшения качества жизни

Выезд специалиста сборка и настройка коляски индивидуально под Вашего ребенка бесплатно

Вертикализатор предназначен для стационарного использования и имеет выдвижной упор для уменьшения габаритов вертикализатора для хранения

Вертикализатор оснащен парой колес для транспортировки, ручки поручней для прохода через межкомнатные двери могут быть убраны с помощью шестигранного ключа

Для настройки под особенности пациента вертикализатор имеет широкий диапазон регулировок по отдельности для каждой стопы и колена по трем направлениям с фиксацией с помощью шестигранного ключа. Сформованные подушки упоров, обеспечивают комфорт и надежность эксплуатации. Страховочные ремни для стоп входят в стандартное оснащение. Вертикализатор оснащен поясным упором с регулировкой по высоте и глубине.

Каждому покупателю колясок для детей с ДЦП мы выдадим всю необходимую для предоставления в ФСС документацию. Это позволит вам получить компенсацию финансов, затраченных на приобретение средства передвижения.

С помощью Aidflex X3 можно проводить лечение следующих заболеваний

  • Инсульт
  • Черепно-мозговая травма
  • Спиномозговая травма
  • Инфекции и интоксикации ЦНС
  • Полирадикулонейропатии
  • Рассеянный склероз
  • Нижний парапарез
  • Гемипарез

Особенности Aidflex X3 BASIC

  • Грудной упор - Регулируется по высоте и глубине
  • Рукоятка - Для перемещения
  • Поручни стандартные - Для проемов шириной 1 метр, снимаются с помощью шестигранного ключа
  • Настройка коленоупора - С помощью шестигранного ключа
  • Привод - Макс. вес пользователя 110 кг
  • Регулировка стоподержателей - С помощью шестигранного ключа
  • Колеса - Для перемещения без пользователя
  • Съемный упор - Позволяет более комфортно хранить вертикализатор (не мешает подъезду на кресло-коляске)

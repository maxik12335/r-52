Ходунки для инвалидов Ormesa Dynamico (Динамико)

Тренажер ходьбы инвалидов (ходунки) Динамико - тренажер ходьбы инвалидов (ходунки) Динамико для использования на улице и в помещении для детей и взрослых

Выезд специалиста сборка и настройка коляски индивидуально под Вашего ребенка бесплатно

Каждому покупателю колясок для детей с ДЦП мы выдадим всю необходимую для предоставления в ФСС документацию. Это позволит вам получить компенсацию финансов, затраченных на приобретение средства передвижения.

Особенности конструкции Динамико

  • Абдуктор, отверстия для его монтажа
  • Рукоятки для регулировки высоты поддержки подмышек и таза
  • Отверстия для установки регулируемого руля
  • Мягкая поддержка подмышек регулируются по высоте. Может быть удалена для людей с достаточным контролем вертикального положения
  • Направляющие для ступенчатой регулировки поддержки подмышек и таза
  • Подвеска для поддержки таза регулируется по высоте или вперед-назад для стимулирования ходьбы барицентрическим движением вперед. Регулировку облегчает открытая задняя часть. Легко снимаются для стирки
  • Утяжелители, отверстия для установки
